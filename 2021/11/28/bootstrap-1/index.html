一、响应式布局

引入
  • 同一套页面可以兼容不同分辨率的设备。

实现∶依赖于栅格系统∶将一行平均分成12个格子，可以指定元秦占几个格子

步骤∶

  1. 定义容器。相当于之前的 table

    • 容器分类(样式)

      • container

        固定宽度

      • container-fluid

        自适应宽度

  2. 定义行。相当于之前的 tr

    • 样式：row

  3. 定义元素。指定该元素秦在不同的设备上，所占的格子数目

    • 样式：col-设备代号-格子数目
      • xs : 超小屏幕 （<768px）
      • sm : 小屏幕平板(≥768px)
      • md :中等屏幕桌面显示器(>992px)
      • lg :大屏幕大桌面显示器(≥1200px)
